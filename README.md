# Tic Tac Toe

A simple Tic Tac Toe game wrapped with Electron for desktop.

## Features
- Play Tic Tac Toe in a desktop app
- Simple, dependency-only setup via npm

## Install

```bash
npm install
```

## Run (development)

```bash
npm start
```

## Build (Windows)

```bash
npm run package-win
```

## Notes
- The `dist/` and `node_modules/` folders are excluded from the repository.

## Project description
Tic Tac Toe is a simple two-player game implemented with HTML, CSS, and JavaScript, packaged into a desktop application with Electron. The app supports local play on a single machine.
